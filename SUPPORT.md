# Support

The fastest way to get useful help is to share a tiny redacted reproduction.

## Bug Reports

Open a bug report with:

- your `envgap --version`
- your Python version
- a small `.env`
- a small `.env.example`
- the Python snippet that reads env vars
- the envgap output

Please redact API keys, tokens, passwords, and connection strings.

## Questions

Use GitHub Discussions for questions, project shapes, roadmap ideas, and "should envgap detect this?" conversations.

## Feature Requests

The best feature requests describe a repeated config failure, not just an implementation idea. A real-world example helps a lot.

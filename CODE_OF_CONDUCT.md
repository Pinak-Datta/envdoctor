# Code of Conduct

envgap follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Our Standard

Please be kind, direct, and practical. Good faith disagreement is welcome. Harassment, personal attacks, and spam are not.

## Scope

This applies to project spaces including issues, pull requests, discussions, release comments, and community threads connected to envgap.

## Reporting

If something needs attention, open a private report to the maintainer through GitHub or email the maintainer listed on the project profile.
